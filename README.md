# Daily Revision Quiz

Year 8 adaptive revision quiz for Maths & Science.

## Topics
- **Maths:** Algebra, Bearings, Compound Shapes, Circles, Angles, Probability,  Percentages
- **Science:** Forces, Electromagnetism, Work Done, Periodic Table, Chemical Reactions

## Features
- 18 MCQs per quiz (randomly selected)
- Auto-scoring with topic breakdown
- Adaptive difficulty (focuses on weak areas next time)
- Mobile-friendly UI

## Run locally
```bash
npm install
npm start
```

## Deploy on Render
1. Connect this repo on render.com
2. Build Command: `npm install`
3. Start Command: `npm start`
